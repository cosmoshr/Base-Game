## Running

### To run in webpack dev mode

```sh
yarn && yarn serve
```

### To run in electron dev mode

yarn && yarn serve:electron

## Building

```sh
yarn && yarn build
```

### To build for electron

```sh
yarn && yarn build:electron
```
